JEWELRY SETS AUTO-CATALOG
=========================

Upload jewelry-sets.json to the ROOT of ImageKit.
The page reads:
https://ik.imagekit.io/k0wpvuatq/jewelry-sets.json

Each jewelry set uses exactly three images:
<slug>-1.png
<slug>-2.png
<slug>-3.png

Examples:
maharani-manika-gulabi-set-1.png
maharani-manika-gulabi-set-2.png
maharani-manika-gulabi-set-3.png

To add a new set:
1. Upload its three PNG images to ImageKit root.
2. Add one object to jewelry-sets.json and re-upload that JSON file.

Example object:
{
  "slug": "new-royal-set",
  "name": "New Royal Set",
  "description": "Description shown beneath the carousel.",
  "price": 89,
  "extension": "png"
}

A browser cannot safely list all files in an ImageKit account from the public CDN.
The JSON catalog is therefore required for automatic product discovery without exposing a private ImageKit API key.
