For Maharani Manika Gulabi Set to work in AR exactly like Royal Drops,
upload these separate transparent PNG overlays to the root of ImageKit:

maharani-manika-gulabi-set-ar.png
maharani-manika-gulabi-set-necklace-ar.png
maharani-manika-gulabi-set-left-earring-ar.png
maharani-manika-gulabi-set-right-earring-ar.png

The website now checks for the three separate overlay files and will use them for
necklace + left earring + right earring placement. If they are missing, the page
shows a helpful status message instead of broken image icons.
