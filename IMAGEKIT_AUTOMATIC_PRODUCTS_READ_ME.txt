GLOBAL RANI — AUTOMATIC JEWELRY CARDS

1. In ImageKit, open the folder named global-rani-products.
2. For each new jewelry set, upload three files with the same beginning.

Example:
maharani-manika-emerald-set.png
maharani-manika-emerald-set-ar.png
maharani-manika-emerald-jewelry-box-opening.gif

3. Optional: on the MAIN image, add ImageKit custom metadata:
productName = Maharani Manika Emerald Set
description = A regal emerald-hued jewelry set for festive celebrations.
price = 65
category = Jewelry Set
active = true

4. Upload this entire website to GitHub and wait for Netlify to say Published.
5. Test: https://YOUR-SITE.netlify.app/api/jewelry-products
6. Refresh the Jewelry Sets page with Ctrl + F5.

The website automatically creates the gold card and connects View Set, Add to Cart, and Try It On.
